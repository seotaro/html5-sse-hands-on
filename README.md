# HTML5 SSE（Server-Sent Events） Hands-on

サーバーでは`fastify-sse-v2`プラグインを使用した。
クライアントのデモページもホストする。

## Install

```bash
yarn
```

## Run

```bash
node main.js
```

クライアントページ <http://localhost:8080/>
